# Developer Notes on Claude Code Usage

This folder will be at `.claude.rstsr` directory, which will be tracked by git. You can symlink it to `.claude` if you want.

This project excludes `.local`-like paths, and you can add local configurations to it, without polluting the main repository.

Anyway, this Claude Code workflow is still in early stages, may change frequently and not that good to work. You can always write your own `.claude`.

This folder can be regarded as a temporary developer documentation.
